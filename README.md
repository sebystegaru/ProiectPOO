# Proiect 5 – Platformă pentru Gestiunea Taskurilor într-o Echipă (Project Management)
## Descriere aplicație
Se va dezvolta o aplicație de tip project management simplificat, unde echipele pot crea proiecte, pot asigna taskuri membrilor și pot urmări progresul.
Aplicația trebuie să includă cel puțin două tipuri de utilizatori (de exemplu: manager de proiect și membru echipă), cu autentificare obligatorie.
## Managerul de proiect
Managerul creează și organizează proiectele.
### Funcționalități:
1. Administrarea proiectelor
  - poate crea, modifica și șterge proiecte;
  - poate configura detalii precum deadline, prioritate, descriere.
2. Gestionarea taskurilor
  - poate crea taskuri și subtaskuri;
  - poate asigna taskuri către membri ai echipei.
3. Monitorizarea progresului
  - poate vedea statusul fiecărui task (to-do, în progres, finalizat);
  - poate genera rapoarte despre progresul proiectului.
4. Gestionarea membrilor echipei
  - poate adăuga sau elimina membri din proiecte;
  - poate vizualiza workload-ul fiecărui membru.
## Membrul echipei
Membrul lucrează la taskurile asignate.
### Funcționalități:
1. Vizualizarea taskurilor asignate
  - poate vedea toate taskurile proprii organizate pe proiecte.
2. Actualizarea statusului
  - poate marca taskurile ca în progres sau finalizate;
  - poate adăuga comentarii și actualizări.
3. Dashboard personal
  - poate vedea taskurile prioritare și deadlinurile apropiate.
4. Colaborare
  - poate lăsa comentarii la taskuri pentru comunicare cu echipa.
